# Spring Core and Maven Exercises

This directory contains the implementations for the Spring Core and Maven enabler exercises, organized as a multi-module Maven project.

## Project Structure

The project is structured into three submodules to separate and showcase different Spring Configuration and Bootstrap styles:

1. **[LibraryManagement_XML](file:///c:/Users/bened/Downloads/Misson_Cognizant/Week_2/SpringCoreMaven/LibraryManagement_XML)**
   - Covers Exercises 1, 2, 3, 4, 5, 7, and 8.
   - Demonstrates traditional Spring XML configuration, dependency injection (constructor and setter injection), and AspectJ-based AOP logging.
2. **[LibraryManagement_Annotations](file:///c:/Users/bened/Downloads/Misson_Cognizant/Week_2/SpringCoreMaven/LibraryManagement_Annotations)**
   - Covers Exercise 6.
   - Re-implements dependency injection and bean definition using annotation-based configuration (`@Service`, `@Repository`, `@Autowired`) with component scanning.
3. **[LibraryManagement_SpringBoot](file:///c:/Users/bened/Downloads/Misson_Cognizant/Week_2/SpringCoreMaven/LibraryManagement_SpringBoot)**
   - Covers Exercise 9.
   - Bootstraps the Library Management service using a standard Spring Boot 3 framework.

## Exercises Covered

- **Exercise 1**: Configuring a Basic Spring Application
- **Exercise 2**: Implementing Dependency Injection
- **Exercise 3**: Implementing Logging with Spring AOP
- **Exercise 4**: Creating and Configuring a Maven Project
- **Exercise 5**: Configuring the Spring IoC Container
- **Exercise 6**: Configuring Beans with Annotations
- **Exercise 7**: Implementing Constructor and Setter Injection
- **Exercise 8**: Implementing Basic AOP with Spring
- **Exercise 9**: Creating a Spring Boot Application
